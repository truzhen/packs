# 软件说明书

本文说明 Pack 的使用场景、对象、流程、能力需求和安全边界。模板内容作为作者填写示例，确保工程结构的规范化与权属确立。

## 1. 软件概述

本软件是一款专为 Truzhen V3 平台设计和定制开发的 **“企业级长周期项目管理 Scene Pack 软件”**。软件以声明式规格和受控适配代码为核心，主要用于对企业长周期项目（如装修施工、业务推进、合规证据链等）的业务实体建模、DAG 流程推进及安全门禁管理。

## 2. 软件运行环境

- **支撑平台**：Truzhen V3 场景流程引擎 (Scene Flow Engine) L2 层
- **宿主系统**：macOS / Linux / Windows 10+
- **底层依赖**：Truzhen Runtime, GORM (SQLite Pure Go), Go 1.22+, pnpm 9+
- **必需网关**：Model Gateway, Communication Gateway, Execution Gateway

## 3. 功能模块结构

1. **数据建模模块**：提供基于 JSON Schema 的强类型 Business Object 规格定义与 optimistic concurrency control (OCC) 乐观锁版本字段支持。
2. **流程推进调度模块**：基于有向无环图 (DAG) 的 SceneFlowRun 状态转移与自动 DAG 自愈注入（缺失 Terminal Nodes 自动首尾自愈）。
3. **主权安全防卫模块**：配合平台 `Base Gate`、`SideEffectGate` 保证任何真实发送、真实执行或写正式 Memory 前，均需主人 (Owner) 进行 OwnerDecision 终裁。
4. **适配隔离与 UI 视图模块**：声明外部软件（如 Matrix/Dendrite 聊天或 AutoCAD）的接口契约与 Mock 仿真，并且可定制在 Truzhen 标准 slots 下的 declarative UI surface slots 投影。

## 4. 特色与安全承诺

- **权属清晰**：本说明书所述的业务剧本、对象 Schema、UI 视图及原创适配代码归属于 Pack 开发者，具备独立著作权条件。
- **纯粹无害**：默认不联网、不调用真实宿主 API，拒绝携带任何 API Key 和敏感凭证明文。
- **真实工程**：包含完整的编译脚手架与红队单元测试，可作为真实生效的软件工程作品交付。
