# 软件版本说明书 (Release Notes)

记录版本变更、兼容性、迁移提示和升级规则。

## 1. 当前版本信息

- **当前版本**：v1.0.0 (Gold Master)
- **发布日期**：2026-05-30
- **适配平台**：Truzhen V3 Platform (L2/L3)

## 2. 新增特性与优化

1. **多对象并发支持**：彻底实现 `BusinessObject` 实体、`SceneFlowRun` 实例与 `PackSpec` 图纸的三权分离，同款 Pack 能够同时高效安全推进多个项目而不交叉污染。
2. **DAG 自动自愈设计**：即使 Pack 页面视觉设计中遗漏了 Start/End 节点，转换器亦能进行 Permissive Node Mapping 并自动在流程首尾注入 `auto_start` 和 `auto_end`，保证项目能够完美开始 and 圆满归档。
3. **主权隔离策略落地**：实现静态 manifest 声明，所有外部软件对接逻辑默认锁定 Mock 状态，杜绝第三方安全隐患。

## 3. 版本兼容性与升级规则

- **升级规则**：
  - PackSpec 发布后应保持 immutable，版本变更必须有明确审计。
  - 新版本不得自动影响正在运行 of 正在运行的 SceneFlowRun，不同运行实例互不干扰。
  - 迁移必须生成候选，并经 OwnerDecision 和 Receipt 记录。
- **兼容性声明**：
  - 本版本与 Truzhen V3 的底层 Storage、Receipt Ledger 及 Scene Flow 模块完美适配。
  - 升级到新版 Pack 时，老版本正在运行的项目不受影响，充分维护用户长期数据的稳定运行。
