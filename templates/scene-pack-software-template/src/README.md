# 辅助业务逻辑层 (src/)

本目录包含本 Scene Pack 作者自写的校验器 (Validators)、状态转换器 (Transforms)、渲染器 (Renderers) 等强类型辅助业务逻辑代码。不得读取 raw secret 或绕过 Gateway。

## 📋 目录原则

1. **真实有效的逻辑单元**：我们坚决反对和禁止为了软著拼凑毫无意义的垃圾代码或伪造的混淆代码。在此处编写的 TS/JS/Go 代码应是本项目用于进行本地 Schema 数据合法性校验与结构映射的真实功能代码。
2. **生成的代码独立**：SDK 自动生成的类型定义及脚手架应放置在 `generated/` 目录中。在此处的代码是由 Pack 作者纯手工原创编写的，拥有绝对的独立知识产权 (IP)。
3. **无状态设计 (Stateless) 与仿真用途**：此处逻辑应当设计为纯函数，避免读写全局共享的可变状态 (mutable global state) 或强侵入宿主全盘，所有输入均来自参数，输出均由 return 返回。本目录代码只能服务于作者工程内的静态检查、数据转换和本地仿真。运行态是否采用这些逻辑，由 Pack Studio 审查后转换为安全规格或候选适配声明。
